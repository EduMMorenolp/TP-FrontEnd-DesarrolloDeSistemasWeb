# Estilo Visual

## Direccion estetica

- Tema RPG / gremio de aventureros.
- Tono epico, pero legible y funcional.
- Portada con combate visual contra el Bug.
- Páginas individuales con identidad de personaje.

## Decisiones a definir

## Decisiones aplicadas en la base actual

- Paleta base implementada en `Proyecto/css/style.css`:
	- Fondo principal: `#f5efe4`
	- Superficie de tarjetas: `#fffaf1`
	- Tinta principal: `#2f1f14`
	- Acento principal: `#a13f2a`
	- Acento secundario: `#d07a2d`
- Header oscuro para jerarquia visual y contraste de navegacion.
- Tarjetas con borde suave y sombra ligera para separar bloques de contenido.
- Boton principal con estado hover visible para reforzar interactividad.

## Responsive aplicado

- Breakpoint `1200px`: ajuste de ancho del contenedor.
- Breakpoint `900px`: centrado de header y navegacion.
- Breakpoint `400px`: enlaces de navegacion apilados a ancho completo.

## Pendientes de cierre visual

- Definir tipografias finales de Google Fonts para cumplir consigna.
- Ajustar identidad visual individual por integrante sin romper coherencia global.
- Documentar iconografia definitiva y criterio de uso en README.

## Registro obligatorio

Cada eleccion visual debe documentar:
- por que se eligio,
- donde se aplica,
- que beneficio aporta a la lectura o navegacion.
